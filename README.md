💳 GameMart Pay – UPI Payment Gateway UI (Simulation)

A modern, high-performance UPI Payment Gateway UI Simulation built using pure HTML, CSS, and JavaScript.
This project replicates a real-world fintech payment flow including amount input, QR-based payment, and transaction confirmation.

«⚠️ Disclaimer: This is a frontend UI simulation project. It does NOT process real payments.»

---

🚀 Live Demo

👉 https://github.com/S2zxx0zxx/GameMart-Payment-Gateway/

✨ Features

- 💰 Dynamic Amount Input
  
  - Real-time validation (₹1 – ₹2000 limit)
  - Quick amount selection (₹50, ₹100, ₹500, etc.)

- 📊 Payment Limit Indicator
  
  - Visual progress bar for transaction limits

- 📱 QR Code Payment UI
  
  - Scan & Pay simulation
  - UPI ID display and copy feature

- ⏱️ Live Timer System
  
  - QR expiration countdown
  - Refresh payment option

- 📤 Payment Proof Upload
  
  - Upload screenshot simulation
  - Drag & drop support

- ✅ Transaction Success Screen
  
  - Animated success state
  - Auto-generated transaction receipt

- 🎨 Premium UI/UX
  
  - Glassmorphism + neon theme
  - Smooth animations & transitions
  - Fully responsive design

⚙️ How It Works

1. User enters payment amount
2. System validates input and enables payment
3. QR code is displayed for payment simulation
4. User uploads payment proof or marks as paid
5. System verifies (simulated) and shows success screen
